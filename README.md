# 📝 Smart To-Do List Pro

## 📌 Project Description

Smart To-Do List Pro is a web-based task management application built using HTML, CSS, and JavaScript. It helps users organize daily tasks efficiently by allowing them to add, manage, and track tasks with priorities and due dates. The system provides a clean interface and automatically updates task status.

---

## ❗ Problem Statement

Managing daily tasks manually can be confusing and inefficient. Users often forget deadlines or lose track of completed tasks. This project solves the problem by providing a simple and interactive digital solution to organize and track tasks effectively.

---

## 🚀 Features

### 🔹 Core Features

* Add tasks
* Delete tasks
* View all tasks
* Mark tasks as completed

### 🔹 Advanced Features

* Priority levels (High / Medium / Low)
* Due date selection
* Save and load tasks using localStorage
* Display pending vs completed tasks
* Task counter (Total / Completed / Pending)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## ▶️ How to Run the Project

1. Download or clone the repository
2. Open the `index.html` file in any web browser
3. Start managing your tasks

---

## 🎯 How to Use

1. Enter task name
2. Select priority level
3. Choose due date
4. Click **"Add Task"**
5. Mark tasks as completed or delete them
6. View task statistics
